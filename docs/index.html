Leibniz Programming Language

Created by Cameron, Atul, Doug, and Artur

About the Language

The Leibniz programming language is designed to enhance mathematical functionality and readability. Inspired by Leibniz, the co-creator of calculus, our goal is to make mathematical computation more intuitive and accessible.

Leibniz supports function calls, function creation, for loops, print statements, variable declarations, object declarations, and conditional statements. It includes built-in math functions like domain, cos, and pi.

The language aims to make math enjoyable and easy to integrate into code, even for those who may struggle with it.
